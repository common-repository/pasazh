Sync your Woocommerce website with Pasazh. export your products to Pasazh and keep them sync with each other
== Description ==
This addon help you to connect your Woocommerce website to Pasazh. Pasazh is an Iranian Marketplace and you can import
your products to Pasazh from your website. Just install and enable it. Then in your Pasazh dashboard type your website
url and press "Connect"
== Installation ==
This section describes how to install the plugin and get it working.
e.g.
1. Upload `pasazh` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
